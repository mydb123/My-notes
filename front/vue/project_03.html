# router-link的用法

  1. <router-link :to="{path:'/test',query: {name: id}}">跳转</router-link>(id是参数)使用：this.$route.query.id

  2. <router-link :to="'/test/'+id">跳转</router-link>(id是参数)

注意：router-link中链接如果是‘/’开始就是从根路由开始，如果开始不带‘/’，则从当前路由开始
