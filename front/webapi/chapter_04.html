# 本地存储

# Window.sessionStorage

  1. 数据纯存在浏览器页面
  2. 页面刷新不会丢失数据
  3. 可以存5M
  • 存储数据
     sessionStorage.setItem(key,value)
  • 获取数据
    sessionStorage.getItem(key)
  • 删除
    sessionStorage.removeItem(key)
  • 清楚所以数据
    sessionStorage.clear();

# Window.localStorage

  1. 生命周期永久生效,除非手动删除
    localStorage.setItem(key,value)
  • 获取数据
    localStorage.getItem(key)
  • 删除
    localStorage.removeItem(key)
  • 清楚所以数据
    localStorage.clear();

# 节点操作

  • 父节点parentNode(得到的是最近的父节点)
    <ul class="box">
        <li class="erwiitem">x</li>
        <li></li>
        <li></li>
        <li></li>
    </ul>
    <script>
        // var father = document.querySelector(".box")
        var erzi = document.querySelector(".erwiitem")
        //亲爸爸
        console.log(erzi.parentNode);
    </script>
  • 子节点children(常用)
    var father = document.querySelector(".box")
    // var erzi = document.querySelector(".erwiitem")
    console.log(erzi.children);
  • 返回第一个和最后一个子元素(常用)
    var father = document.querySelector(".box")
    // var erzi = document.querySelector(".erwiitem")
    //返回第一个
    console.log(children[0]);
    //返回最后一个
    console.log(children[father.children.length - 1]);
